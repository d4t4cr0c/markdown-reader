# Markdown Reader

Welcome to the **Markdown Reader** desktop application!

## Features

- Read and render Markdown files beautifully
- Light and Dark mode support
- Clean, distraction-free reading experience

## Supported Markdown Elements

### Text Formatting

You can use **bold**, *italic*, or ***bold and italic*** text.

### Lists

Unordered list:
- Item 1
- Item 2
- Item 3

Ordered list:
1. First item
2. Second item
3. Third item

### Code

Inline code: `const hello = "world";`

Code block:
```javascript
function greet(name) {
  console.log(`Hello, ${name}!`);
}
```

### Blockquotes

> This is a blockquote.
> It can span multiple lines.

### Links

Visit [GitHub](https://github.com) for more information.
